# DATA 221 – Assignment 2

This repository contains my solutions for **DATA 221: Introduction to Data Science – Assignment 2**.

# Contents
The repository includes Python scripts for each question, along with required input and output files:

- `question1_assignment2_data221.py` – Word frequency analysis on a text file  
- `question2_assignment2_data221.py` – Bigram frequency analysis  
- `question3_assignment2_data221.py` – Detection of near-duplicate lines  
- `question4_assignment2_data221.py` – Filtering student data and saving results  
- `question5_assignment2_data221.py` – Creating grade bands and grouped summaries  
- `question6_assignment2_data221.py` – Crime risk categorization and unemployment comparison  
- `question7_assignment2_data221.py` – Web scraping page title and first paragraph  
- `question8_assignment2_data221.py` – Web scraping section headings  
- `question9_assignment2_data221.py` – Web scraping tabular data and saving to CSV  
- `question10_assignment2_data221.py` – Reusable function for searching text files
